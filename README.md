
# About Chatting App

Uses Socket.io for real time communication and stores user details in encrypted format in Mongo DB Database.
## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB

## Run Locally

Clone the project

```bash
  git clone git@github.com:luuthai2k/App_Chat.git
```

Go to the project directory

```bash
  cd App-Chat
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```
Start the Client

```bash
  cd frontend
  npm start
```

That's all.
